POC to use JSR 354 : Money and Currency API

Scope
====
To create a small trading application to show the dashboard with 2 currencies USD and INR

Initially the data is taken from a file and which has the values in INR.


Design
=====
Have a currency provider to get the data from file and provide as the Currency objects

Have a converter to currencies vice versa.

Have a formatter to show the formated data in a dashboard.
